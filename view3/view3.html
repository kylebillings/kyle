Responsive Web

Prepared a phototography driven music discovery platform for launch. Implemented UI on a previously coded cakephp backend.

Project Scope

For Ekko.fm, I was brought on to add branding elements, interactions, and onboarding content to an existing web application. Music and photo uploads were already set up, and I organized the assets into a more modern, intuitive layout.


Landing Page

Starting with the landing page, I located imagery that suggested both photography and music, the two main content types for the site. This would become the account creation screen after the login wall was removed.

Before

After


After creating an account, the application delivered users to their new (empty) feed. To clear up confusion, I added a conditional that presented a bit of onboarding copy to first timers with empty feeds. I also consolidated navigation, gave more real estate to the discover tab, and filed activity, upload, and profile into a single My Ekko page.
